# Agenda SPA

## Project Description
Agenda SPA is a single-page application designed for managing agendas, providing a user-friendly interface for tracking events and appointments.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/rdtimana/agendaspa.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd agendaspa
   ```
3. **Install dependencies**:
   ```bash
   npm install
   ```
4. **Run the application**:
   ```bash
   npm start
   ```
5. **Open your browser** and go to `http://localhost:3000` to view the application.
